# AI-Movie-Recommendation-App
A Machine Learning based Movie Recommendation System built with Python, Streamlit, and TMDB API that suggests similar movies with posters.

# 🎬 Movie Recommender System

A Machine Learning based Movie Recommendation System that suggests movies similar to the one selected by the user.

Built using:

* Python
* Streamlit
* Pandas
* Machine Learning
* TMDB API

---

## 🚀 Features

* Recommend similar movies instantly
* Fetch movie posters using TMDB API
* Interactive Streamlit UI
* Fast recommendation using similarity matrix

## ⚡ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/Movie-Recommender-System.git
```

Move into the directory:

```bash
cd Movie-Recommender-System
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
